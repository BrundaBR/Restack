## About flutter 
1.native cross platform ,one code base for both ios,android
2. everything is a widget.
3. How is it transformed to native app?
    -dart code is converted to native code by flutter sdk for both android and ios machine 
4.Folders
    -.idea-for vscode config
    -android- Flutter sdk when converts to native code , if the system is android then it will be built for android device and source code is in this folder.
    -build -  output of flutter project
    managed by flutter sdk
    -ios- ios source code
    -lib - working folder 
    -test - to write tests
    -pubspec.yaml - manage third party packages and to add images ,text fonts,bascically config file
